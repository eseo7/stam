요구사항 가져오기 및 산출물 초안 생성 6개 게시판은 기본으로 존재합니다. 요구사항 표 데이터를 가져오면 요구사항 게시판 데이터가 생성되고, 이를 기준으로 기능정의·메뉴/화면·WBS·화면설계서·테스트 시나리오 초안이 생성됩니다. Local IndexedDB No Firebase Import Trigger
이 화면은 무엇을 하나요? 이 화면은 요구사항 표 데이터를 가져와 6개 게시판의 초안 데이터를 자동 생성하는 흐름을 검증합니다. 게시판은 이미 존재하며, 가져오기는 게시판을 만드는 것이 아니라 데이터 생성 자동화입니다. 각 게시판에는 엑셀 없이 직접 하나씩 등록할 수도 있으며, 가져오기는 자동화를 위한 선택 흐름입니다. 모든 자동 생성 데이터는 확정이 아니라 초안 / 검토 필요 상태로 만들어지고, 원본 요구사항과 연결됩니다.
입력 방식 안내 엑셀 파일을 업로드하면 첫 번째 시트의 헤더/행을 읽어 요구사항 가져오기 입력값으로 변환합니다. 이번 단계는 1차 시트/기본 헤더 기준입니다. 복잡한 병합 셀·다중 시트·고급 컬럼 매핑은 후속 단계에서 처리합니다. 외부 CDN/라이브러리 없이 브라우저 내장 기능만으로 .xlsx 를 파싱하며, 샘플 데이터 불러오기 또는 TSV/CSV 붙여넣기도 그대로 사용할 수 있습니다.
▸ ⓪ 요구사항 엑셀 업로드 (선택) Excel

엑셀 파일을 업로드하면 첫 번째 시트의 헤더/행을 읽어 요구사항 가져오기 입력값으로 변환합니다.
이번 단계는 1차 시트/기본 헤더 기준입니다. 복잡한 병합 셀·다중 시트·고급 컬럼 매핑은 후속 단계에서 처리합니다.

지원 헤더: requirementId · title · description · priority · actor · requirementType · sourceNote (한글 헤더 요구사항ID/제목/설명/우선순위/사용자/유형/비고 등도 1차 매핑).

▸ ① 요구사항 표 입력 Input

입력 컬럼: requirementId · title · description · priority · actor · requirementType · sourceNote (탭 또는 콤마 구분, 첫 줄은 헤더). requirementId · title 은 필수입니다.
