기능 내용

기능 설명
요구사항정의서 목록 화면에서 등록된 모든 요구사항을 조회하는 기능입니다. 검색 및 필터 조건에 따라 목록을 동적으로 갱신합니다.
입력 조건
검색어(기능 ID, 기능명, 담당자), 필터 조건(상태, 유형, 우선순위, 담당자)
처리 규칙
1. 전체 요구사항 목록을 기본 정렬(최종 수정일 내림차순)로 표시한다. 2. 검색어 입력 시 ID·이름·담당자에 대해 부분 일치 필터링을 수행한다. 3. 필터 패널 적용 시 선택된 조건과 AND 조건으로 필터링한다.
예외/오류 처리
검색 결과가 없을 경우 "조건에 맞는 항목이 없습니다" 메시지를 표시한다.
