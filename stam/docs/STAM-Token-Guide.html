역할: 사람용 토큰 파생 뷰

STAM Token Guide is not the SSOT. The SSOT for token values is stam/css/stam.tokens.css. This guide is a derived human-readable reference.

토큰 이름·의미·사용 원칙·상태/우선순위/관계 톤 기준을 설명합니다. 현재값 확인은 항상 stam/css/stam.tokens.css를 따릅니다.

01 Purpose Derived Reference
  • stam/css/stam.tokens.css에 정의된 디자인 토큰을 사람이 읽기 쉬운 형태로 정리한다.
  • 토큰 이름, 의미, 사용 원칙, 상태·우선순위·관계 톤 기준을 문서화한다.
  • 승인된 Color Delta 변경 이력을 기록한다 — 현재값 SSOT가 아니다.
  • 제품 화면·문서 작업 시 하드코딩 색상 대신 var(--token-name) 사용을 강제한다.
02 SSOT Rule 필수
SSOT Token values are defined only in stam/css/stam.tokens.css. This guide must not be treated as the authoritative source for current hex or rgba values.
  • 토큰 추가·변경은 stam/css/stam.tokens.css PR에서만 수행한다.
  • 이 가이드는 파생 문서이므로, CSS와 불일치 시 CSS가 우선한다.
  • 에이전트·구현 작업 전 반드시 CSS 파일에서 현재값을 확인한다.
03 Token Usage Principles 기준 고정
  • 색상·간격·radius·폰트는 --* 토큰만 사용한다.
  • CSS에서 var(--token-name) 형태로 참조한다.
  • 문서·가이드 본문에 현재값 hex를 반복 기재하지 않는다.
  • 임의 토큰명 생성·하드코딩 색상·px 직접 기입은 금지한다.
  • 라이트/다크 대응은 [data-theme="dark"] 블록의 토큰 재정의를 따른다.
