원칙 · 정책

STAM 보드 목록에서 행 클릭과 체크박스 클릭은 역할이 완전히 분리됩니다.
이 기준은 요구사항정의서 동작을 표준으로 정의한 것입니다.
  • 행 클릭 → 행 selected 상태 + 좌측 bar + 상세 Drawer 열림
  • 체크박스 클릭 → 체크박스 checked 상태만 변경 (이벤트 전파 차단)
  • 두 동작은 절대 교차하지 않습니다
01

정상 행 클릭 동작

두 번째 행이 클릭된 상태입니다. 체크박스는 선택되지 않았지만 selected bar가 표시됩니다.
ID요구사항명상태
RQ-001로그인 화면승인완료
RQ-002회원가입 화면검토중
RQ-003메인 대시보드작성중
정확한 구현 패턴
/* 행 클릭 핸들러 */ row.addEventListener('click', function (e) { // 체크박스 셀 클릭이면 무시 if (e.target.closest('.checkbox-cell')) return; // 기존 selected 해제 document.querySelectorAll('.data-row').forEach(function (r) { if (r !== row) r.classList.remove('selected'); }); // 현재 행 selected 추가 (체크박스 checked 변경 없음) row.classList.add('selected'); // 상세 Drawer 열기 openDrawer('detail'); });
02

체크박스 클릭 동작

체크박스를 클릭하면 checked 상태만 변경됩니다. 행 selected 상태나 Drawer 열림에 영향을 주지 않습니다.
/* 체크박스 클릭 — 이벤트 전파 차단 필수 */ checkbox.addEventListener('click', function (e) { e.stopPropagation(); // row click으로 전파되지 않음 }); checkbox.addEventListener('change', function () { // 체크박스 상태만 처리 (row selected 변경 없음) updateDeleteButton(); }); /* 체크박스 td도 전파 차단 (안전 장치) */ checkboxCell.addEventListener('click', function (e) { e.stopPropagation(); });
03

헤더 체크박스 동작

헤더 체크박스는 전체 체크/해제만 담당합니다. 행 selected 상태를 변경하거나 Drawer를 열지 않습니다.
headerCheckbox.addEventListener('change', function () { var checked = headerCheckbox.checked; document.querySelectorAll('.row-checkbox').forEach(function (cb) { cb.checked = checked; // row.classList.toggle('selected', ...) — 금지 ❌ // openDrawer() — 금지 ❌ }); updateDeleteButton(); });

05

금지 패턴

❌ 행 클릭 시 체크박스 자동 선택
row click 핸들러에서 checkbox.checked = true 또는 checkbox.checked = false를 호출하는 것은 금지입니다.
체크박스 상태는 오직 체크박스 직접 클릭으로만 변경됩니다.
❌ 체크박스 클릭 시 상세 Drawer 열림
체크박스 click/change 핸들러에서 openDrawer()를 호출하면 안 됩니다.
반드시 e.stopPropagation()으로 row click 전파를 차단해야 합니다.
❌ selected row인데 bar 없음
행이 selected 상태(클릭)일 때 좌측 3px bar가 반드시 표시되어야 합니다.
CSS에서 첫 번째 td에 position: relative가 있어야 ::before bar가 보입니다.
❌ updateMasterChk()를 row click 핸들러에서 호출
row click은 체크박스 상태를 변경하지 않으므로 header checkbox 상태를 재계산할 필요가 없습니다.
updateMasterChk()는 checkbox change 핸들러에서만 호출합니다.
