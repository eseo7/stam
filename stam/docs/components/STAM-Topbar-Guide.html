상태: Skeleton — 구조만 생성된 상태입니다. 상세 규칙은 후속 작업에서 확장합니다.
01 목적 Skeleton
  • STAM 화면에서 사용하는 App Topbar와 Doc Nav의 구조 · 역할 기준을 정의한다.
  • 상세 규칙은 후속 작업에서 확장한다.
02 적용 대상 Skeleton
  • App Topbar — 모든 제품 화면 상단
  • Doc Nav — 가이드 · 문서 페이지 상단
03 기본 원칙 Skeleton
  • 높이 --topbar-h: 48px 고정.
  • 좌(로고/브랜드) · 중(타이틀/브레드크럼) · 우(액션) 영역 분리.
  • 테마 토글 버튼 위치는 우측 고정.
04 Do Skeleton
  • 우측 액션 간격 기준을 준수한다.
  • 문서 페이지에서 브레드크럼(상위 문서 링크)을 포함한다.
  • 알림 dot은 Topbar 아이콘에 한해서만 사용한다.
05 Don't Skeleton
  • Topbar 높이 임의 변경 금지.
  • 임의 요소 추가로 Topbar 높이 증가 금지.
  • Left Navigation 메뉴를 Topbar에 연결 금지.
06 QA 체크리스트 Skeleton
  • 높이 48px 고정 확인.
  • 테마 토글 동작 확인.
  • 고정 레이아웃(스크롤 시 고정) 확인.
