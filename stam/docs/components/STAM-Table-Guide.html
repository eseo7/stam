01 목적 Skeleton
  • STAM 화면에서 사용하는 목록/테이블 컴포넌트의 컬럼 · 정렬 · Focus View 기준을 정의한다.
  • 상세 규칙은 후속 작업에서 확장한다.
02 적용 대상 Skeleton
  • B · C 그룹 전 목록 화면
  • WBS 작업 (B3 — 특수 패턴 포함)
  • 오픈 시나리오 (C8 — 특수 패턴 포함)
03 기본 원칙 Skeleton
  • 페이지 레벨 가로 스크롤 금지 — overflow-x: hidden 적용.
  • 테이블 내부 래퍼에만 overflow-x: auto 허용.
  • 컬럼 많은 화면은 Focus View를 필수 제공한다.
04 Do Skeleton
  • 컬럼 많은 화면(B3 · C8 · B9 · B10 · B11)에 테이블 내부 overflow-x: auto를 적용한다.
  • Focus View 버튼을 Page Header 우측에 배치한다.
  • Row 클릭 시 Drawer 상세 진입을 기본으로 한다.
05 Don't Skeleton
  • 페이지 레벨 가로 스크롤 절대 금지.
  • Focus View 없이 컬럼 9개 이상 테이블 제공 금지.
  • 테이블 헤더 임의 생략 금지.
06 QA 체크리스트 Skeleton
  • 페이지 레벨 가로 스크롤 없음 확인.
  • Focus View 진입 · 종료 동작 확인.
  • 컬럼 정렬 동작 확인.
