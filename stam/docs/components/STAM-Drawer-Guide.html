상태: Skeleton — 구조만 생성된 상태입니다. 상세 규칙은 후속 작업에서 확장합니다.
01 목적 Skeleton
  • STAM 화면에서 사용하는 Drawer 패턴의 너비 · 탭 구조 · 스크롤 기준을 정의한다.
  • 상세 규칙은 후속 작업에서 확장한다.
02 적용 대상 Skeleton
  • 목록 화면 등록 · 상세 · 수정 Drawer
  • 전체보기 전환이 필요한 문서형 산출물
  • B · C 그룹 전 목록 화면
03 기본 원칙 Skeleton
  • 너비는 480–560px 범위에서 콘텐츠 복잡도에 따라 결정한다.
  • 탭 기본 구조 5개 — 기본정보 · 댓글/의견 · 첨부파일 · 변경이력 · 관련회의록.
  • Overlay 배경 클릭 시 닫기를 지원한다.
04 Do Skeleton
  • 수정 모드는 상세 Drawer 내부에서 전환한다 (별도 페이지 금지).
  • 탭 구조 5개를 유지한다.
  • 내부 스크롤은 Drawer 콘텐츠 영역에서 처리한다.
05 Don't Skeleton
  • Drawer 중첩 금지.
  • 별도 페이지 전환으로 Drawer를 대체 금지.
  • Drawer 외부 클릭 차단 금지 (닫기 허용 필수).
06 QA 체크리스트 Skeleton
  • Overlay dim 동작 확인 (rgba(23,23,25,.45)).
  • 내부 스크롤 처리 확인.
  • 탭 전환 동작 확인.
