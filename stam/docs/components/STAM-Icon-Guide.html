상태: Skeleton — 구조만 생성된 상태입니다. 상세 규칙은 후속 작업에서 확장합니다.
01 목적 Skeleton
  • STAM 화면에서 사용하는 Icon의 유형 · 크기 · 색상 기준을 정의한다.
  • 상세 규칙은 후속 작업에서 확장한다.
02 적용 대상 Skeleton
  • 버튼 내 아이콘 · 단독 아이콘 버튼
  • 상태 표시 아이콘 · 내비게이션 아이콘
  • 테이블 · 목록 내 인라인 아이콘
03 기본 원칙 Skeleton
  • Outline / Fill 구분 사용 — 일반 상황 Outline, 활성/강조 상황 Fill.
  • 크기는 16 · 20 · 24px 고정 사용.
  • 색상은 currentColor 상속으로 처리한다.
04 Do Skeleton
  • 인터랙티브 아이콘에 aria-label을 추가한다.
  • 일반 컨텍스트에 Outline, 활성/선택 상태에 Fill을 사용한다.
  • 한 행(row)에서 Outline / Fill 혼용을 의도적으로만 허용한다.
05 Don't Skeleton
  • 임의 크기 변경 금지.
  • fill / stroke 색상 직접 지정 금지 — currentColor 사용.
  • 아이콘 대신 유니코드 글리프 사용 금지.
06 QA 체크리스트 Skeleton
  • 인터랙티브 아이콘 aria-label 유무 확인.
  • 크기 기준(16/20/24px) 준수 확인.
  • Outline / Fill 혼용 의도 확인.
