PR #299 범위 확인

  • PR #299는 디자인 Preview만 생성한다.
  • Firebase SDK, Auth Provider, apiKey, authDomain은 이번 PR에서 추가하지 않는다.
  • 실제 Google 로그인 구현은 후속 Auth bootstrap PR에서 진행한다.
  • users/{uid} 생성/갱신, emailNormalized 기반 초대 확인, pending → active 연결은 후속 PR 범위다.
  • 프로젝트 진입 후 기본 landing은 Project Overview로 한다.
