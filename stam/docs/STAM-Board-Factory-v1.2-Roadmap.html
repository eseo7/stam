5. 계약 초안

Data Adapter

  • list(query) → {rows,total}
  • get(id) → record (detail hydration 필수)
  • summary(query) → {metrics,facets} (summary strip 필수)
  • create / update / remove
  • mode: static | localStorage | api | firestore — PR #136 preview는 static

Reference Resolver

  • listOptions(type, query) · resolve(type, ids)
  • type: requirement | screen | screenSpec | wbs | user
  • relation 저장값은 id 또는 {type, ids:[]}, user는 userId 권장

Built-in Renderer Registry

checkbox idName text date chip priorityChip statusChip typeChip user relationChip link actionButtons → 전 컬럼 흡수(custom 0)

Built-in Detail Section

infoGrid textBlock relationCards historyList reviewList acceptanceList approvalStatus attachmentList
