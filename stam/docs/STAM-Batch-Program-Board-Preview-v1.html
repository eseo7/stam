Section 01 · Summary
  • 목적 — 배치프로그램 관리 게시판의 목록·상세·등록·수정 화면이 어떤 구조와 흐름을 가질지 정적 Preview 로 합의한다.
  • Definition 문서와의 관계 — Batch Program Board Definition v1 에서 정의한 목적·컬럼·필드·상태값·연결 산출물을 화면 형태로 구체화한 후속 문서다. Definition = "무엇을", Preview = "어떻게 보일지".
  • 본 문서는 구현이 아니다 — 아래 모든 화면은 .pv 점선 박스 안의 정적 mock 이며, 실제 제품 화면·App Shell·라우팅·데이터 로직과 무관하다. 구현은 별도 PR(page 생성 단계)에서 진행한다.
