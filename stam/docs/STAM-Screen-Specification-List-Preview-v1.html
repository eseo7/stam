UI 상태 샘플
Empty검색 결과 없음조건을 변경하거나 새 문서를 등록하세요.
Loading문서 목록 불러오는 중권한 확인 후 목록을 표시합니다.
Error권한 없음이 문서를 열람할 권한이 없습니다. 관리자에게 문의하세요.
Conflict Candidate저장 충돌 감지 후보처리 기준은 Save & Conflict Handling Policy Guide v1 참고. 충돌 차단 여부는 정책에 따라 결정.
Preview 정책 기준 안내
  • 저장 / 충돌 처리는 Save & Conflict Handling Policy Guide v1 기준을 따릅니다.
  • 변경이력 / 복구는 Change History / Recovery Policy Guide v1 기준을 따릅니다.
  • DB / 권한 / 로그 항목은 모두 Candidate(후보) 단계이며, 정식 스키마 / ERD가 아닙니다.
  • 기술 엔진(에디터, Export 등)은 후보 단계입니다. Screen Specification Full Stack Module Guide v1을 참고하세요.
  • 이 Preview는 기능 구현이 아닌 UX 기준 시안입니다. Firebase / API 연동 없음.
