01 Purpose — 이 문서의 목적 Preview Baseline

이 문서는 STAM 의 주요 진입 기능인 “요구사항 엑셀 가져오기” 흐름을 정적 UX Preview 로 정의한다. 사용자가 기존 요구사항 엑셀을 업로드하면 STAM 이 컬럼을 분석하고, 표준 필드와 customFields 로 매핑한 뒤, 요구사항 게시판 초안으로 가져오는 전체 흐름을 시각화한다. 실제 Excel parsing / Firebase / Storage / DB 작업은 포함하지 않는다.

  • 이 Preview 는 후속 실제 구현의 기준 화면으로 사용한다.
  • Excel Import Builder 는 부가기능이 아니라 STAM 의 주요 진입 기능이다.
  • 업로드 결과는 확정 데이터가 아니라 Draft Import 다.
  • 매핑 · 타입 · 옵션값 · 무시 여부는 사용자 검토와 사람 승인을 거친다.
  • 원본 추적을 위한 source traceability 메타를 전제로 한다.
Preview 핵심 메시지 8가지 ① Excel Import Builder 는 부가기능이 아니라 STAM 의 주요 진입 기능이다. ② 엑셀 업로드 결과는 확정 데이터가 아니라 Draft Import 다. ③ STAM 은 컬럼을 자동 분석하고 표준 필드 매핑을 제안한다. ④ 표준 필드에 맞지 않는 컬럼은 customFields 후보로 분류한다. ⑤ 사용자는 매핑 / 타입 / 옵션값 / 무시 여부를 검토하고 확정한다. ⑥ import 후 요구사항 게시판에는 “초안 / 미검토 / Excel Import 출처” 상태로 들어간다. ⑦ 원본 추적을 위해 sourceFile / sourceSheet / sourceRow / importBatchId / importedAt / importedBy 메타가 필요하다. ⑧ 이 Preview 는 실제 Excel parsing 구현이 아니라 UX 기준이다.
02 User Scenario — 실무 시나리오 Scenario

SI 현장에서 가장 흔한 시작점은 고객사가 보내는 요구사항 엑셀이다. 아래 흐름을 기준 시나리오로 한다.

고객사 요구사항 엑셀 수신→ 기획자가 STAM 업로드→ 헤더·컬럼 자동 분석→ 표준 필드 매핑 제안
고객사 고유 컬럼 = customFields 후보→ 사용자 검토·확정→ 요구사항 게시판 Draft Import 생성
표준 필드로 매핑되는 예
  • 요구사항명 → title
  • 상세내용 → description
  • 우선순위 → priority
  • 상태 → status
  • 담당자 → owner
customFields 후보가 되는 예
  • 고객검토여부 → customFields.customerReviewStatus
  • 1차오픈범위 → customFields.phase1Scope
  • 법무검토필요 → customFields.legalCheckNeeded
  • 레거시코드 → customFields.legacySystemCode
핵심 STAM 은 고객사 고유 컬럼을 버리지 않는다. 표준 필드에 맞지 않는 컬럼은 customFields 후보로 분류해 사용자가 채택 / 병합 / 무시를 결정하게 한다. 최종 생성물은 모두 Draft Import 상태로 들어간다.

5 데이터 검증→ 6 미리보기→ 7 가져오기 확정

13 Guardrails — 가드레일 Guardrails
  • 자동 import 결과는 확정 데이터가 아니다 (Draft Import).
  • 표준 필드 우선 매핑을 기본으로 한다.
  • customFields 는 후보 검토 후 생성한다.
  • 중복 ID 를 방지한다.
  • 필수값(title 등) 누락을 방지한다.
  • source traceability 는 필수다.
  • importBatch 단위 rollback / 비교는 후속 확장 범위다.
  • 기존 요구사항 화면을 대수술하지 않는다.
  • 실제 DB schema 를 이 단계에서 확정하지 않는다.
핵심 가드레일 Draft Import · 표준 필드 우선 매핑 · customField 후보 검토 · 중복/필수값 검사 · source traceability 필수 · 기존 화면 보존 · DB schema 확정 보류.
