1.5. 행 상태 — normal · is-active · is-selected · is-active.is-selected
공통 룰 — .is-active : 행 클릭으로 drawer 가 열린 행 (checkbox 미선택, delete count 미포함, left bar 표시) · .is-selected : checkbox 선택 (bulk action 대상, delete count 포함, left bar 표시) · 두 상태는 독립적으로 동작하며 동시에 부여될 수 있음.
상태 설명 checkbox left bar delete count
normal 기본 행 unchecked 없음 미포함
is-active 행 클릭 / drawer target unchecked 표시 미포함
is-selected checkbox 선택 / bulk action target checked 표시 포함
is-active + is-selected 선택된 행을 클릭해 drawer open checked 표시 포함
