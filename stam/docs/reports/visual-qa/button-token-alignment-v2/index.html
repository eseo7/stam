성격: token alignment — alias 클래스는 보존하면서 토큰 값/참조만 정렬.
PR #133 감사 요약 — Primary 버튼은 light 에서는 --stam=--btn-primary-bg=#5451E8 으로 일치하나 dark 에서 --stam=#8B7FF8 vs --btn-primary-bg=#5451E8 으로 어긋났습니다. Ghost 는 alias 가 --t3, 공통은 --btn-ghost-text(=--t1). Outline hover 는 alias 가 brand-purple 강조, 공통은 neutral. Del 은 alias 가 outline-danger 패턴, 공통은 solid red. 본 PR 은 Primary 다크 토큰 mismatch 만 해결하고, 나머지 의도된 차이는 주석으로 명시한 채 보존합니다. alias 클래스 자체는 제거하지 않습니다.
Browser QA 시도 결과 (follow-up) — 본 환경에는 chromium/chrome/firefox 시스템 바이너리가 없으며, npx playwright install chromium 도 네트워크 egress 제한(cdn.playwright.dev 403 Host not in allowlist)으로 설치 실패했습니다. 따라서 실제 헤드리스 캡처 / console 수집은 본 PR 환경에서 수행 불가합니다. light/dark QA · console errors 는 수동 QA 요청 (browser unavailable) 로 표기하며, 점검 대상 / 토큰 변경 결과는 아래 "Impact Scope" 표를 그대로 사용하면 됩니다.
