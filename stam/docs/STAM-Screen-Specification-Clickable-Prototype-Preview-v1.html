Screen Specification Clickable Prototype Preview v1
화면설계서 목록 → 템플릿 작성 → Register Drawer → Preview → Detail Drawer 흐름을 검증하는 정적 프로토타입
Static Preview No DB/API Template Based Clickable Flow
이 문서는 STAM 화면설계서 작성 UX의 기준본입니다. Register Drawer는 작성, Preview는 검수, Detail Drawer는 저장 결과 확인 역할로 분리합니다.
템플릿 선택은 화면설계서 작성의 필수 시작 단계입니다. 첨부파일은 화면 구조를 대체하지 않고 참고자료로만 사용합니다.
버튼 클릭은 화면 상태 전환만 수행하며 DB/API/저장/권한 로직은 포함하지 않습니다.
