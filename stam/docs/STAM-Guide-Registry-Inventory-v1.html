01 STAM Guide Registry / Inventory v1 Registry

Guide relationship, status, product impact, and next-action register

Registry Docs Control v1 Draft

STAM의 모든 Guide / Applied Preview / Policy 문서를 관리하는 기준 문서이다. 실제 제품 구현 문서가 아니라, 가이드 관계·상태·영향도·후속 작업을 추적하는 관리대장이다.

신규 기능 요청 시 어떤 가이드를 먼저 수정해야 하는지 판단하고, Parent / Related / Docs Index 위치 / 함께 수정할 기존 Guide를 기록하는 원칙을 따른다. 본 문서로는 stam/pages/**, stam/js/**, stam/css/**, stam/assets/**를 수정하지 않는다.

02 Core Principle 필수
  • 기능 요청 1건은 단일 화면 작업이 아니다.
  • 기능 요청 1건은 정책 + DB Impact 후보 + Guide + 화면 + 로직 + 권한 + 로그 + QA + 문서 현행화로 연결된다.
  • 정식 DB 테이블/컬럼 확정은 주요 화면/기능 요구가 충분히 만족된 후 진행한다.
  • 지금 단계에서는 각 Guide에 DB Impact 후보만 기록한다.

03 Status Definition 상태값
Planned

생성 예정 / 아직 문서 없음

Draft

작성 중 / 요구사항 변경 가능성 높음

Candidate

기준 후보 / 실제 반영 전 검토 필요

Stable

현재 적용 기준 / 화면 반영 가능

Locked

변경 제한 / 변경 시 별도 승인 필요

Deprecated

더 이상 기준으로 사용하지 않음
