G. 스펙 요약
  • Input: height --fc-h (36px) / radius --fc-radius (8px) / bd --fc-bd / bg --fc-bg / text --fc-text / 13px
  • Focus: border --fc-bd-focus + box-shadow --fc-ring (0 0 0 3px rgba(stam,.15))
  • Placeholder: color --fc-ph (--t3)
  • Disabled: opacity --fc-disabled-opacity (.45) + pointer-events none + cursor not-allowed
  • Readonly: bg --fc-bg-ro (--bg-sur2) + color --t2 (disabled와 구분)
  • Textarea: min-height 80px / resize vertical (disabled·readonly에서 none)
  • Error: border --fc-err-bd (#EF4444) + bg --fc-err-bg + ring --fc-err-ring + 메시지 .stam-field-error
  • Success: border --fc-ok-bd (#10B981) + bg --fc-ok-bg + ring --fc-ok-ring
  • Warning: 이번 PR에서는 별도 토큰 없이 .is-error 톤을 amber(#F59E0B)로 변수 오버라이드 (제품 적용 보류)
  • Label: 11px / 600 / --t2 · .req 빨간 별표 · .opt 회색 보조
  • Help / Error 메시지: .stam-field-help (t3) / .stam-field-error (err-text + 경고 아이콘)
  • Field-level 상태 전파: .stam-form-field.is-error / .is-success / .is-disabled / .is-readonly 가 내부 control 에 자동 적용
