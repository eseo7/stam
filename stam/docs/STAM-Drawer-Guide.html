원칙 · 정책

STAM Drawer는 보드 화면의 등록/상세/수정 패널을 위한 공통 Slide-in Panel 기준입니다.
--stam-drawer-width 값 하나로 모든 보드 화면의 Drawer 폭이 동시에 조정됩니다.

적용 원칙
  • 기존 page-level class(rq-drawer, msl-drawer)는 유지하고 stam-drawer를 병행 추가
  • CSS 로드 순서: stam.components.css → stam.form-controls.css → stam.drawer.css → 페이지 CSS
  • 페이지 CSS의 width는 min(var(--stam-drawer-width), calc(100vw - var(--stam-drawer-gutter)))로 교체
  • Drawer JS 동작(open/close/ESC/scrim click)은 수정하지 않음
  • footer 구조 통일은 이번 작업 범위 외
01

CSS 변수 목록

--stam-drawer-width 860px 기본 Drawer 폭 (핵심 기준값)
--stam-drawer-width-narrow 720px 좁은 Drawer (.is-narrow)
--stam-drawer-width-wide 1040px 넓은 Drawer (.is-wide)
--stam-drawer-gutter 32px 뷰포트 좌측 여백 (overflow 방지)
--stam-drawer-z 50 Drawer z-index
--stam-drawer-scrim-z 49 Scrim z-index
--stam-drawer-head-px 24px Header 좌우 padding
--stam-drawer-body-px 24px Body 좌우 padding
--stam-drawer-foot-px 24px Footer 좌우 padding

04

적용 방법

① CSS 로드 순서
<link rel="stylesheet" href="../../css/stam.components.css"> <link rel="stylesheet" href="../../css/stam.form-controls.css"> <link rel="stylesheet" href="../../css/stam.drawer.css"> <!-- 추가 --> <link rel="stylesheet" href="../../css/stam.requirements.css"> <!-- 페이지 CSS -->
② HTML class 병행 추가
기존 class는 유지하고 공통 class를 추가합니다.
<!-- 변경 전 --> <div class="rq-scrim" id="rq-scrim"></div> <div class="rq-drawer" id="rq-dw-register">...</div> <!-- 변경 후 --> <div class="rq-scrim stam-drawer-scrim" id="rq-scrim"></div> <div class="rq-drawer stam-drawer" id="rq-dw-register">...</div>
③ 페이지 CSS width 교체
페이지 CSS에서 하드코딩 width를 공통 변수로 교체합니다.
/* 변경 전 */ .rq-drawer { width: clamp(640px, 48vw, 720px); ... } /* 변경 후 */ .rq-drawer { width: min(var(--stam-drawer-width), calc(100vw - var(--stam-drawer-gutter))); ... }
05

DevTools 검증 방법

DevTools Elements 패널 → <html> 또는 :root 선택 → Styles 탭에서
--stam-drawer-width 값을 변경하면 요구사항정의서·메뉴구조화면목록 양쪽 Drawer 폭이 동시에 바뀌는지 확인합니다.
/* DevTools Console에서도 가능 */ document.documentElement.style.setProperty('--stam-drawer-width', '1000px'); /* 원복 */ document.documentElement.style.removeProperty('--stam-drawer-width');
